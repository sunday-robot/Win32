# 説明
.netプログラムからWin32APIを呼び出すための各種定義集

# クラスの構成
|クラス名|説明|
|---|---|
|Constant&nbsp;&nbsp;&nbsp;&nbsp;|定数定義|
|Struct|構造体定義|
|Delegate|コールバックをdelegateで定義|
|Function|関数のdllimport宣言|
|Util|ユーティリティ。<br/>本来はこのようなものは作りたくなかったが、GetLastErrorでエラーの種類を取得し、例外として投げるものだけは面倒すぎるので作成した。|

# 作成ルール
必要になったら各種定義を逐次追加する。<br/>
追加する際は、何も考えずに辞書順になるように並べる。<br/>

DLL名(kernel.dllなど)でグルーピングすることも考えたが、
おそらく後になってグルーピングのルール見直しなどしたくなってしまうと思われたので、
そうならないようにこうした。
