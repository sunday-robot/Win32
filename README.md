# 説明
.netプログラムからWin32APIを呼び出すための各種定義集

# クラスの構成
|クラス名|説明|
|---|---|
|Win32Constant&nbsp;&nbsp;&nbsp;&nbsp;|定数定義|
|Win32Struct|構造体定義|
|Win32Delegate|コールバックをdelegateで定義|
|Win32Function|関数のdllimport宣言|
|Win32Util|ユーティリティ。<br/>本来はこのようなものは作りたくなかったが、GetLastErrorでエラーの種類を取得し、例外として投げるものだけは面倒すぎるので作成した。|

# 保守方針
必要になったら各種定義を逐次追加する。<br/>
Utilへの追加は原則行わない。

# ルール
追加する際は、何も考えずに辞書順になるように並べる。<br/>
DLL名(kernel.dllなど)でグルーピングすることも考えたが、
おそらく後になってグルーピングのルール見直しなどしたくなってしまうと思われたので、
そうならないようにこうした。
